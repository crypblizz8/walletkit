# React Native Wallet Kit

Bootstrap Web3 Wallet Components with one simple package.

## Installation

```sh
yarn add walletkit
```

## Usage

```js
import { WalletKitProvider } from 'walletkit';
...

 <WalletKitProvider projectId={projectId} relayUrl={relayURL} metadata={metadata}>
    <Component>
 </WalletKitProvider>
```
